## 终点

按用户给定需求完成本地 Compose UI 工作流：底部导航按“扫描 / 查看平台列表 / 查看卡槽 / 查看标签”组织；扫描页、平台列表、卡槽、标签页各自只展示对应内容；所有详情和待标注流程隐藏底部导航且不残留透明触击阻挡层；外部 LLM 自动标注暂不实现。

## 已关闭范围

- 外部 LLM 自动标注不做。
- 标签创建/合并优先复用现有平台标签事实源。若现有数据层缺完整标签实体或合并事务，本轮用 UI 状态和明确提示表达“已创建/将按该名称合并集合”的本地交互，不引入迁移。

## 功能票据

1. 导航与页面隔离：重命名并重排底部导航，详情/待标注/创建页隐藏底部导航，隐藏时移除底部栏占位和触击层。
2. 扫描页精简：只保留扫描状态、扫描统计、权限降级提示和扫描操作按钮。
3. 平台列表与详情：平台页只展示搜索框、固定首项“待标注平台”和平台列表；支持模糊动态筛选；平台详情只有详情信息、平台短信证据列表和返回。
4. 待标注短信：独立页面隐藏底部导航；显示剩余数量、短信卡片、选择标注按钮、可搜索单选标签列表、悬浮确认、最终确认标注并进入下一条。
5. 卡槽页：只展示卡槽 1、卡槽 2、未知卡槽大卡片及短信数量；卡槽详情展示卡槽信息和对应平台列表；平台详情复用平台页交互。
6. 标签列表与创建/合并：标签页只展示标签列表、搜索按钮、创建按钮；搜索激活后支持动态筛选和首项快速创建；创建页支持名称创建和合并已有标签的搜索、多选、确认与回填。
